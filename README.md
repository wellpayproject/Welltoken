# WELL Token
WELL is a fixed-supply ERC-20 utility token deployed on the Base network.

- Network: Base
- Standard: ERC-20
- Total Supply: 100,000,000 WELL (fixed)
- Minting: No additional minting possible
- Taxes / Fees: None

WELL is designed to support future on-chain utilities, ecosystem integrations, and transparent value flow.
